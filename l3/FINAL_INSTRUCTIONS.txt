### ШАГ 2: Загрузка на GitHub (10 минут)

**Выполнить в PowerShell/Command Prompt по одной команде:**

```bash
# 1. Переход к папке
cd c:\Users\plcds\Desktop\l3

# 2. Инициализация Git
git init

# 3. Конфигурация (замените на ваши данные)
git config user.name "Your Full Name"
git config user.email "your.email@gmail.com"

# 4. Добавление всех файлов
git add .

# 5. Проверка что добавилось
git status

# 6. Первый коммит
git commit -m "Laboratory Work #3: Complete UML conceptual and logical design

- Created directory structure: 2-SoftwareDesign, 1.2.3-UseCaseDiagram, 1.4.1-SoftwareArchitectConcept
- Implemented all UML diagrams: Use Case, Deployment, Concept Classes
- Created relational and JSON data models
- Developed comprehensive SQL schema with constraints and indices
- Added attribute vocabulary documentation
- Included code examples and implementation guidelines"

# 7. На GitHub: создайте новый репозиторий https://github.com/new
# Затем выполните следующие команды:

git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git branch -M main
git push -u origin main
```

**ВНИМАНИЕ: Замените:**
- `Your Full Name` → ваше имя
- `your.email@gmail.com` → ваш email
- `YOUR_USERNAME` → ваш GitHub username (например: plcds)
- `YOUR_REPO` → название репозитория (например: software-design)
