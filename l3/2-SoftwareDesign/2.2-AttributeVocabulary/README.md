### Словник атрибутів об'єктів

| Об'єкт | Атрибут | Короткий опис | Тип | Обмеження |
|--------|---------|---------------|-----|-----------|
| Person | name | Прізвище та ім'я особи | String | не порожнє, мах 100 символів |
| Person | age | Вік особи | int | > 0 та <= 150 |
| Teacher | course | Предмет, який викладає | String | не порожнє, мах 100 символів |
| Teacher | workExperience | Кількість років роботи | int | >= 0 |
| Student | course | Назва курсу, що вивчається | String | не порожнє, мах 100 символів |
| Student | studentGroup | Номер групи студента | String | не порожнє, мах 50 символів |
| Teacher | students | Список студентів, що навчаються | List<Student> | може бути порожнім |
| Student | teacher | Викладач студента | Teacher | обов'язковий |
